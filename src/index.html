🚦 Traffic Controller

Control the lights. Avoid crashes. Survive!

🎮 How to Play

  1. Edit player.js to write your traffic control algorithm
  2. Refresh this page to test your changes
  3. Keep your score above zero to survive
  4. ⚠️ WARNING: Ambulance crashes = instant game over!

🚗 Vehicle Types

  • 🚑 Ambulance - Highest priority!
  • 🚓 Police - High priority
  • 🏛️ Government - Medium priority
  • 🚗 Regular - Standard

💡 Tips

  • Check for active crashes before changing lights
  • Use isIntersectionSafe() helper
  • Prioritize emergency vehicles
  • Don't make vehicles wait too long
  • Read README.md for full API docs
